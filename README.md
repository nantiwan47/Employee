### การติดตั้งและรันโปรเจกต์

#### 1. Clone โปรเจกต์
```bash
git clone https://github.com/nantiwan47/Employee.git
cd Employee
````

#### 2. ติดตั้ง Dependencies

```bash
flutter pub get
```

#### 3. รันแอป

* **บน Chrome (Web)**

```bash
flutter run -d chrome
```

* **บน Emulator หรือ Device**

```bash
flutter run
